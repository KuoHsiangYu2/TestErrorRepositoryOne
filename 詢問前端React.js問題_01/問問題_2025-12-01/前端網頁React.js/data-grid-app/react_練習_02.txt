安裝完 nvm 後，你可以使用以下指令安裝你所需的 Node.js 版本：
>> nvm install 版本號


例如，如果你想要安裝 Node.js 的 LTS 版本，可以執行：
>> nvm install --lts


安裝完多個版本後，你可以隨時切換到所需的版本：
>> nvm use 版本號
>> nvm use 14.17.6


列出所有已安裝的 Node.js 版本：
>> nvm ls


查看當前使用的 Node.js 版本：
>> nvm current


>> nvm install --latest-npm
這將安裝最新版本的 npm（Node 包管理器）。


>> nvm upgrade
這會升級 nvm 到最新版本。


>> npm install
安裝 node_modules


>> npm  run XX
執行 package.json XX指令
